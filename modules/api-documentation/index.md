          <h2 id="module-overview">Module Overview</h2>
          <p>API documentation is the cornerstone of developer experience. This module covers best practices for creating clear, comprehensive documentation that serves both new and experienced developers. You'll learn how to structure documentation, write for different audiences, and create content that helps users succeed with your API.</p>
          
          <h3>Learning Objectives</h3>
          <ul>
            <li>Understanding documentation types</li>
            <li>Creating reference documentation</li>
            <li>Writing tutorials and guides</li>
            <li>Documenting code samples</li>
            <li>API documentation tools</li>
            <li>Documentation testing</li>
            <li>Maintaining documentation</li>
          </ul>
        </div>
