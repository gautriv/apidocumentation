          <h2 id="module-overview">Module Overview</h2>
          <p>This module serves as an introduction to API documentation, covering the basics of how the internet works, what APIs are, and the role of technical writers in creating API documentation. By the end of this module, you'll have a solid understanding of the fundamentals of API documentation.</p>
          
          <h3>Learning Objectives</h3>
          <ul>
            <li>Understand how the internet works at a high level</li>
            <li>Learn what APIs are and why they're important</li>
            <li>Identify what technical writers document in APIs</li>
            <li>Understand the process of documenting APIs</li>
          </ul>
