          <h2 id="module-overview">Module Overview</h2>
          <p>API documentation tools are crucial for creating professional, comprehensive, and maintainable documentation. This module explores various tools, platforms, and frameworks that streamline the documentation process—from generating reference documentation to creating interactive experiences for API consumers. You'll learn about specialized documentation tools, frameworks, static site generators, and techniques for creating documentation that evolves alongside your API.</p>
          
          <h3>Learning Objectives</h3>
          <ul>
            <li>Understanding the core features of modern API documentation tools</li>
            <li>Mastering OpenAPI and Swagger tooling</li>
            <li>Creating interactive API documentation</li>
            <li>Implementing documentation as code workflows</li>
            <li>Working with specialized API documentation platforms</li>
            <li>Integrating documentation tools with development workflows</li>
            <li>Automating documentation generation</li>
          </ul>
